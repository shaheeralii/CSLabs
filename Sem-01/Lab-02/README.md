# Lab 02: Control Flow & Decision Making

This lab focuses on decision-making structures, logical branching, and input validation in C++.

## 📋 Task Overview

| File | Task | Description |
| :--- | :--- | :--- |
| `task1_input_finder.cpp` | **Character Classifier** | Detects if an input is a digit, vowel, or consonant using relational logic. |
| `task2_simple_shop_cart.cpp`| **Shopping Cart** | A retail system that applies a 10% discount for totals exceeding 15,000 Rupees. |
| `task3_age_analysis.cpp` | **Age Comparison** | Finds the youngest of three friends to determine activity priority. |
| `task4_reservation_sys.cpp` | **Ticket Reservation** | Validates seat availability and updates inventory for a 35-seat bus. |
| `task5_find_weekday.cpp` | **Weekday Finder** | Uses a `switch-case` statement to map integers (1-7) to days of the week. |
| `task6_medical_diagnosis.cpp`| **Medical Diagnosis** | Categorizes body temperature into Hypothermia, Normal, or Fever ranges. |
| `task7_stdgrade_calc.cpp` | **Grade Calculator** | Evaluates student performance (A-F) based on numerical marks with range validation. |

## 🧠 Key Concepts
* **Conditional Branching:** Implementing complex `if-else if-else` ladders.
* **Logical Operators:** Using `&&` (AND) and `||` (OR) to check multiple conditions simultaneously.
* **Switch-Case:** Efficiently handling multi-way branching for menu-driven tasks.
* **Input Validation:** Ensuring data falls within logical ranges (e.g., marks between 0-100).
