# Lab 01: Programming Fundamentals

This folder contains C++ programs focused on basic arithmetic, constant handling, and the implementation of simple financial and time-based logic.

## 📋 Task Overview

| File | Task | Description |
| :--- | :--- | :--- |
| `task1_miles_to_km.cpp` | **Miles to KM** | Converts miles to kilometers using constant conversion factors. |
| `task2_bank_sys_if_else.cpp` | **Banking (If-Else)** | A simple banking system handling transactions using `if-else` branching. |
| `task2_bank_sys_switch.cpp` | **Banking (Switch)** | The same banking system logic implemented using a `switch-case` structure. |
| `task3_day_calculator.cpp` | **Day Calculator** | Calculates days or date-related logic based on user input. |
| `task4_billing_system.cpp` | **Billing System** | Generates bills by calculating item totals and applying relevant math logic. |
| `task5_time_converter.cpp` | **Time Converter** | Transforms total seconds into a readable Hours:Minutes:Seconds format. |
| `task6_guessing_game.cpp` | **Guessing Game** | A simple interactive number guessing game using basic conditional logic. |

## 🧠 Key Concepts
* **Multiple Implementations:** Comparing `switch` vs `if-else` for the same logic (Bank System).
* **Financial Logic:** Managing balances and calculating totals (Billing/Banking).
* **Math Operators:** Extensive use of division `/` and modulus `%` (Time Converter).

## 🚀 How to Run
1. Open your terminal.
2. Compile: `g++ <filename>.cpp -o out`
3. Execute: `./out`
