# Lab 03: Iterative Logic & Loops

This repository contains C++ solutions for Lab 03, focusing on advanced loop structures, nested iterations, and string accumulation techniques.

## 📋 Task Overview

| File | Task | Description |
| :--- | :--- | :--- |
| `task1_simple_ascii_loop.cpp` | **ASCII Loop** | Iterates through character codes to display ASCII values. |
| `task2_simple_inv_manage.cpp` | **Inventory Management** | Tracks stock levels using loops and basic arithmetic. |
| `task3_prime_num_add.cpp` | **Prime Adder** | Identifies prime numbers from 10 inputs and calculates their total sum. |
| `task4_score_to_factorial.cpp`| **Score to Factorial** | Calculates factorials for multiple students using nested loops. |
| `task5_table_multiplication.cpp`| **Table Visualization** | Generates multiplication tables using a `do-while` loop. |
| `task6_payroll_calc.cpp` | **Payroll Calculator** | Processes employee wages and displays a grouped summary using string accumulation. |

## 🧠 Key Concepts Covered
* **Nested For-Loops:** Used in prime number detection and factorial calculations.
* **Do-While Loops:** Implemented for menu-driven logic and multiplication tables.
* **Boolean Flags:** Used to track states (like primality) within iterative blocks.
* **String Accumulation:** A non-array technique to store and display multiple results at the end of a program.
* **Data Types:** Utilizing `long long` to prevent overflow in high-value calculations (factorials).
